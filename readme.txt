DUZZ adds meta data for SEO and Facebook shares as well as analytics and webmaster tools code to your site and much more. 

== Description ==
DUZZ SEO is a totally free SEO plugin that enables you to perform several important tasks that you can accomplish only by using other premium SEO plugin available on the internet. This plugin is highly optimized for speed by using native WordPress Functions and Hooks for optimum web-page rendering speed. You can add several meta tags that are highly recommended to boost on page SEO as well as for Facebook sharing, analytics and webmaster codes of different channels and much more. Also within coming soon options, you will be able to check run time on page SEO score of the page or post. You will be adding google map and schema to any post or page using only a short code. We are not going to stop here but this plugin is going to be all you need SEO Plugin at one place. Stay tuned! 

For backwards compatibility, we do not support right now. Only the WordPress latest version is recommended to be used.

DUZZ Features:

1. Adds Meta Data / Meta Tags to boost on page SEO
2. Adds Facebook Meta tags that enables best results while sharing on Facebook
3. Adds Google Analytics and Webmaster Verification and Tracking Code
4. Adds Bing and Pinterest Analytics/Webmaster Verification and Tracking Code

Upcoming Features:

* Adds Google Map anywhere in your website you want
* Adds Schema anywhere in your website you want
* Shows Results of On-Page SEO at run-time
* XML Site map generator
* robots.txt file generator
* Ping tool to ping SE with any post or page

== Installation ==
1. Upload \"duzz-seo\" directory to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. And that\'s it .......... you have installed and ready to use DUZZ.
